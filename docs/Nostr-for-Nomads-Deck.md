### Glossary

**Nostr Terms:**
- **Nostr**: "Notes and Other Stuff Transmitted by Relays" - open protocol for decentralized communication
- **NIP**: Nostr Implementation Possibility - standardized protocol features
- **Relay**: Server that stores and forwards Nostr events
- **Event**: Signed data structure (posts, messages, metadata)
- **npub**: Public key in Bech32 format (your Nostr ID)
- **nsec**: Private key in Bech32 format (your secret, never share)
- **Gift-wrap**: NIP-17 double encryption for private messages
- **Zap**: Lightning Network tip sent via Nostr (NIP-57)

**Bitcoin Terms:**
- **Lightning Network**: Layer-2 payment protocol for instant, low-fee Bitcoin transactions
- **Satoshi (sat)**: Smallest Bitcoin unit (0.00000001 BTC)
- **WebLN**: Browser API for Lightning wallet integration
- **LNURL**: Unified standard for Lightning payments and authentication
- **Invoice**: Lightning payment request

**Platform Terms:**
- **MAU**: Monthly Active Users
- **DAU**: Daily Active Users
- **ARPU**: Average Revenue Per User
- **GMV**: Gross Merchandise Value (total transaction volume)
- **Churn**: Rate at which users stop using the product
